ОТЧЁТ ПО ЛАБОРАТОРНОЙ РАБОТЕ

 Генератор уникальных комбинаций элементов из нескольких последовательностей с применением `map`

## Условие задачи
Создать генератор, который выдаёт все возможные уникальные комбинации элементов из нескольких последовательностей (декартово произведение). Подсчитать общее количество комбинаций. К генератору должна быть применена хотя бы одна из функций `map`, `reduce`, `filter`.

## Описание проделанной работы
Для решения задачи написана программа которая:

1. Использует встроенную функцию `itertools.product` – она возвращает генератор, выдающий все уникальные комбинации (декартово произведение) переданных последовательностей.
2. Применяет функцию высшего порядка `map` к полученному генератору. В качестве преобразующей функции используется лямбда-выражение, которое форматирует каждую комбинацию в строку вида `"элемент1-элемент2-элемент3"`. Это демонстрирует применение `map`.
3. Подсчитывает общее число комбинаций простым перемножением длин исходных последовательностей (без использования `reduce` или `filter`, чтобы подчеркнуть именно использование `map`).
4. Выводит на экран общее количество комбинаций и несколько первых результатов после применения `map` для наглядности.
